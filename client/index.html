2

Extract the Files

Once downloaded, extract the ZIP file to a folder on your computer. Remember the location as you'll need it in the next step.

3

Open Chrome Extensions

4

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the Extensions page.

Click "Load unpacked" button and select the folder where you extracted the Research Sathi files.
